# Omnify Frontend

An Angular web application providing the user interface for the Omnify platform, featuring authentication, account activation, and a modern dark-themed UI.

## Tech Stack
- Angular 17+
- TypeScript
- Bootstrap
- ng-openapi-gen (API client generation)

## Features
- User registration with form validation
- Sign in with JWT authentication
- Account activation via email code
- Responsive dark-themed UI
- Auto-generated API services from OpenAPI specification

## Prerequisites
- Node.js 18+
- npm
- Angular CLI
